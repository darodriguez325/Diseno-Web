LECTURA 3

Estructura visual HTML5


Para comenzar a diseñar sitios web solamente se necesita un editor de texto y un servidor local como Apache, pero debemos enunciar las herramientas que utilizaremos durante el proceso de creación de un sitio web adecuado y fácilmente desarrollable para nosotros. En primer lugar encontramos las herramientas Git y GitHub que serán instalados en nuestro ordenador, además se deben configurar correctamente para poder hacer uso de ellos, GitHub nos brinda la oportunidad de elaborar y editar proyectos colaborativos, además también guarda los repositorios en la nube siendo así públicos o privados para que así se puedan editar en conjunto con el equipo de trabajo, para esto se debe crear una cuenta en la pagina oficial de GitHub, se registra la información personal, se crea un repositorio y se comparte con los compañeros de trabajo y su ventaja mas relevante es que es totalmente gratuito. (Castillo, 2012) En segundo lugar encontramos una herramienta muy practica llamada sublime Text, este es un editor de código multiplataforma, esta diseñado para programar sin distracciones, con un estilo de interfaz muy cómodo y sencillo, este es un software de pago pero de igual forma tiene una versión gratuita para poder probarlo, su interfaz oscura y sus colores que resaltan los textos hacen de este un programa ideal para programación de páginas web.
