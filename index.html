El diseño es fundamental para el éxito de un sitio web, se entiende por diseño la forma y estilo de la página, es decir, no basta con crear una pagina mediante HTML y CSS, sino que la pagina debe resultar útil y atractiva para el usuario.

IMPORTANCIA:
El éxito de un sitio web no depende solamente de que se muestren bueno contenidos, sino que gran parte del éxito se debe al diseño.

Objetivo general:


Dar a conocer la unidad uno de diseños de sitios web, referente a introducción, definición e importancia de esta.

Objetivo específico 1:


Explicar la importancia de diseño de sitios web para estudiantes que cursen el programa de Ingeniería, informática, sistemas o telecomunicaciones.

Objetivo específico 2:


Utilizar en el sitio web diferentes ayudas graficas para complementar la información suplementada, tales como videos o imágenes.

Objetivo específico 3:


Brindar información de diferentes fuentes bibliográficas en temas relacionados a diseños de sitios web
